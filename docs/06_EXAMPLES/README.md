# Baa Examples

This directory contains example Baa programs demonstrating the language features.

## Examples

| File | Description |
|------|-------------|
| [hello_world.baa](hello_world.baa) | Basic Hello World program |
| [variables.baa](variables.baa) | Variable declarations and types |
| [functions.baa](functions.baa) | Function definitions and calls |
| [control_flow.baa](control_flow.baa) | If/else, loops, and control statements |
| [arabic_showcase.baa](arabic_showcase.baa) | Arabic syntax features showcase |

## Running Examples

```bash
# Build the compiler
cd build
cmake --build .

# Run an example
./baa ../docs/06_EXAMPLES/hello_world.baa
```

## Learning Path

1. Start with `hello_world.baa` - Basic program structure
2. Continue with `variables.baa` - Learn types and declarations  
3. Move to `functions.baa` - Define and call functions
4. Explore `control_flow.baa` - Loops and conditionals
5. Finally `arabic_showcase.baa` - Full Arabic syntax features
